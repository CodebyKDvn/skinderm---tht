# Skin Cancer AI Diagnosis (SkinGuard AI)

This project provides an AI-powered system for skin cancer diagnosis through image analysis and model segmentation.

## Project Structure
- `backend/`: FastAPI server, models, and processing scripts.
- `assets/`: Videos and images used in the application.
- `css/`, `js/`: UI/UX assets for the web application.
- `index.html`: Main landing page.
- `app.html`: Core application interface.

## Quick Start
1. Install dependencies:
   ```bash
   pip install -r backend/requirements.txt
   ```
2. Run the backend:
   ```bash
   python backend/main.py
   ```
3. Open `index.html` in your browser.

## Features
- AI Image Analysis (Mole/Lesion classification)
- U-NET Image Segmentation
- AI Doctor Chat Support
- High-Performance Backend (FastAPI)
